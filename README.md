# Mindly – Social Media Platform

Mindly is a full stack social media web application inspired by Twitter/X, built using Django REST Framework and React.

This project demonstrates modern full stack architecture, authentication, REST APIs, and responsive frontend design.

## Features

Implemented:

- User authentication (login/logout)
- JWT authentication
- User profiles
- Create posts
- Feed system
- Like/unlike posts
- Block users
- Responsive layout
- Protected routes
- REST API backend
- PostgreSQL database

In progress:

- Comments
- Notifications
- Image upload improvements
- Deployment improvements

## Tech Stack

Backend:
- Python
- Django
- Django REST Framework
- PostgreSQL

Frontend:
- React
- React Router
- Tailwind CSS

Testing:
- Pytest (backend)
- Cypress (frontend)

## Architecture

Frontend communicates with backend via REST API.

React handles UI and state management.
Django REST handles business logic and database operations.

## Purpose

This project was built to demonstrate full stack development skills using modern technologies.
